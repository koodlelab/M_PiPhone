LapsePiTouch
============

PiPhone - A DIY cellphone based on Raspberry Pi by `David Hunt <http://www.davidhunt.ie>`_.

Based on code by PaintYourDragon (Phil B) for Adafruit Industries.

Read more about this project at `Dave's Blog <http://www.davidhunt.ie/piphone-a-raspberry-pi-based-smartphone/>`_.

Enjoy!
